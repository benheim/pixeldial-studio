# PixelDial Studio Site

Marketing site for **PixelDial Studio** (retro-futurist web agency).

## Stack
- React + Vite
- GitHub Pages deployment via GitHub Actions (`.github/workflows/deploy.yml`)

## Local development
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run preview
```

## Deploy
Deployment is automatic on push to `main` (or manual via workflow dispatch).

Live URL:
- https://benheim.github.io/pixeldial-studio/
